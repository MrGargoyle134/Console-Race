Console Race is a free game for Windows.
Finish the race at first to win!

#HOW TO PLAY
Controls player 1:

RIGHT - Right
LEFT  - Left
UP    - Up
DOWN  - Down

Controls player 2:

D     - Right
Q     - Left 
Z     - Up 
S     - Down

ENJOY!
